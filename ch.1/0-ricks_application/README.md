## Something Went Wrong

Rick started losing customers because the app didn’t work the way it was supposed to.

Every time a customer tried to search for what they wanted, they couldn’t find anything.

The problem was in string comparison.
For example, "Fender" and "fender" were treated as different values because string comparison is case-sensitive.

## First Step: Make It Work
Before improving the design or adding new features, the first priority was to make the application work correctly.
